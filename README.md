# ohmly
Ohmly is an AI-powered household energy benchmarking platform that helps users analyze electricity usage, compare with similar homes, receive personalized energy-saving insights, and generate smart sustainability reports. Built to promote efficient living through data-driven decisions.


Built with an interface-first architecture, Ohmly transforms user inputs into benchmark comparisons, smart recommendations, environmental metrics, and downloadable reports.

---

## 🌍 Problem Statement

Many households pay electricity bills without knowing:

- Whether their usage is efficient or excessive  
- How they compare with similar homes  
- What steps can reduce future bills  
- Their environmental impact in CO₂ terms  

Ohmly converts simple household data into practical energy intelligence.

---

## 🚀 Core Features

### 🏠 Household Energy Benchmarking
Compare a household’s monthly electricity usage with similar anonymized homes.

### 📊 Usage Insights
Clear visual summaries and easy-to-read dashboards that explain household consumption patterns.

### 📈 Efficiency Scoring
Generates an energy efficiency score based on household inputs and benchmark logic.

### ♻️ CO₂ Equivalent Measurement
Estimates the carbon footprint associated with household electricity usage.

### 💡 Recommendations
Provides useful suggestions to reduce wastage and improve efficiency.

### 📄 PDF Report Generation
Downloadable report containing metrics, comparisons, charts, and suggestions.

### 🔒 Privacy Focused
Uses anonymized comparison methods for safer benchmarking.

## 💻 Why HTML Is the Majority of Code

Ohmly is designed as a user-facing product where experience, accessibility, and clean presentation are critical.

- It builds the complete interactive interface users directly use  
- Dashboards, forms, reports, charts layout, and navigation rely on structured frontend pages  
- Strong UI matters in real-world adoption — users must understand insights instantly  
- Fast-loading web interfaces are ideal for hackathon demos and scalable deployment  

This does **not** mean the product is static. HTML forms the structure, while JavaScript powers interactions and Python handles calculations, scoring, benchmarking, and report generation.

In short:

**HTML creates the product experience.  
JavaScript creates responsiveness.  
Python creates intelligence.**

---

## 🧠 How It Works

1. User enters household details and bill information  
2. System processes energy patterns  
3. Usage is compared with similar anonymized homes  
4. Efficiency score is generated  
5. CO₂ footprint is calculated  
6. Recommendations are shown  
7. PDF report is generated instantly
